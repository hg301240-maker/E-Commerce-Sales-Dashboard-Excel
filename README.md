# 📊 E-Commerce Sales Dashboard (Microsoft Excel)

## Overview

This project is an interactive **E-Commerce Sales Dashboard** developed in **Microsoft Excel** to analyze sales performance and generate business insights. The dashboard enables users to monitor key performance indicators (KPIs), identify sales trends, compare regional performance, and evaluate product category profitability through dynamic visualizations.

The project demonstrates practical Excel skills commonly used in business reporting and data analysis.

---

## Project Objectives

* Analyze overall sales performance
* Monitor monthly sales trends
* Compare regional sales performance
* Identify top-performing product categories
* Track profit and revenue metrics
* Build an interactive dashboard for business decision-making

---
